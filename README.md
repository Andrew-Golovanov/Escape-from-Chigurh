# Escape-from-Chigurh
В этой игре вам предстоит сбежать от одного из самых жестоких людей в мире - Антона Чигура. Вы находите чемодан с деньгами, таким образом призвав Антона Чигура. Вы столкнётесь с опасными ситуациями, в которых ВЫ решаете, как поступить.
Чтобы запустить проект, нужно скачать все файлы. Проект запускается из файла main.py. В ходе игры появятся два файла: save.txt и character_name.txt. Первый файл - это сохранение прогресса. Если его удалить после сохранения прогресса, то в главном меню будет плашка "❌ Нет сохранений". Второй файл, это имя персонажа. Если после ввода имени персонажа вы случайно удалите файл с именем, то имя всё равно будет в диалоговых сценах.
Если что-то не так с проектом, то удалите папку _pychache_ (это кэш), либо перезапуситите редактор кода.
Следуйте инструкциям в игре. При неправильном вводе вы просто получите оповещение об отсутствии такого варианта ответа.
Удачи!
